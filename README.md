# JFSL Enterprise Network Design Project

## Overview
This project presents the design and implementation of a secure and scalable enterprise network for Jubilee Financial Services Ltd (JFSL) using Cisco Packet Tracer.

## Key Features
- VLAN-based network segmentation
- Inter-VLAN routing using Layer 3 switches
- OSPF dynamic routing protocol
- DHCP, DNS, HTTP, Email servers
- NAT/PAT for internet access
- Site-to-site IPsec VPN
- VoIP telephony services
- Wireless networking
- Access Control Lists (ACLs) for security

## Network Design
- Hierarchical model (Core, Distribution, Access)
- Multi-floor enterprise setup
- Redundant ISP connections

## Tools & Technologies
- Cisco Packet Tracer
- Networking Protocols (OSPF, VLAN, NAT, VPN)
